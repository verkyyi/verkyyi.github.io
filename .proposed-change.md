# Proposed Change: evolve-config-research-source-direct

## What
Directly update the research source in `state/evolve_config.md` from `verkyyi/agentfolio` to `verkyyi/tokenman`.

## Why
The repository `verkyyi/agentfolio` was renamed to `verkyyi/tokenman`. Every hourly evolve run hits a 301 redirect when checking this source — wasting an API call 24 times/day for 5+ days. Two PRs (#54 and #55) attempted this fix but both merged without actually modifying the config file: #54 only wrote a `.proposed-change.md`, and #55 only deleted that file. This is the 3rd attempt.

## How
Edit `state/evolve_config.md`, line 34:
```diff
 ## Research Sources
-- verkyyi/agentfolio
+- verkyyi/tokenman
 - quarto-dev/quarto-cli
```

## Risk
None. State file update (AUTO tier). The redirect is already confirmed by 50+ evolve runs over 5 days. The old name returns HTTP 301 to the new name.

## Autonomy Tier
AUTO — state file update per CLAUDE.md autonomy rules.
